Propojit Google Drive
Zápisky se budou automaticky ukládat na tvůj Drive — dostupné na mobilu i počítači.
1
Jdi na console.cloud.google.com, vytvoř projekt „PetrsBrain"
2
Povol Google Drive API → Credentials → Create → API Key
3
Také vytvoř OAuth 2.0 Client ID (Web application), přidej svoji doménu do Authorized origins
4
